About the Project:

This web-app aims to remove the various bottlenecks while conducting a blood donation camp and the entire blood donation process. It provides a siingle platform for organisers to conduct blood camp. It also provides a feature for emergency blood requirements where a person can post for help if he cannot find a particular blood type in nearby blood-banks. Intrested volunteers would then contact that person to donate blood. This also aims to improve the blood availability between blood banks.

Emergency blood needs

Here you can post for or search for emergency blood needs.If you need blood immediately but it is not available then post a request here....Any intrested blood donor will contact you directly. If you want to help someone in distress then visit this place to see available requests right now.
